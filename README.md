1.  git init   # 初始化

2. git status   # 检测当前目录下的文件状态，3种状态：

3. 红色：新增的文件、修改过的原始文件 》 git add 文件名

4. 绿色：通过git add 过的文件

5. 个人信息的配置：用户名、邮箱等（在commit之前需要添加个人信息）

   git config --global user.email '444611811@qq.com'

   git config --global user.name 'zw'

6. 生成版本： git commit -m ' 备注、版本号等'

7. git log    # 查看版本提交记录
